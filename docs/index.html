Master Implementation

See the entire ecosystem working in harmony.

ExampleApplication.java The Complete Lifecycle 
@SpringBootApplication
public class AgentApp {

    public void runWorkflow(DagExecutor dagExecutor) {
        // 1. Guardrails: Mask PII from user input
        PiiMasker masker = new PiiMasker();
        String safeInput = masker.mask("Analyze user: upendra.manike@gmail.com");

        // 2. Orchestration: Execute with trace tracking
        String traceId = UUID.randomUUID().toString();
        ExecutionContext context = ExecutionContext.builder()
                .executionId(traceId)
                .build();
        context.setParam("input", safeInput);

        dagExecutor.execute(myDag, context)
                .map(results -> {
                    // 3. Binding: Repair LLM JSON output
                    JsonRepairer repairer = new JsonRepairer();
                    String cleanJson = repairer.repair((String) results.get("node1").getOutput());
                    
                    // 4. Observability: Record trace span
                    TraceSpan span = TraceSpan.builder()
                            .traceId(traceId)
                            .response(cleanJson)
                            .timestamp(Instant.now())
                            .build();

                    return cleanJson;
                }).subscribe();
    }
}

Ecosystem Deep-Dive

Every library, explained.

⚡

Orchestrator Core

Multi-agent workhorse that executes non-blocking workflows. Handles parallel nodes, retry policies, and execution context preservation.

💾

Advanced Memory

A session state engine moving data between Redis (Hot) and Vector DBs (Cold), with automatic context window compression.

🛡️

Safety Guardrails

Compliance layer for redacting PII, masking sensitive data in logs, and providing role-based response filtering.

📸

AI Observability

分布式 tracing for LLM prompts, token cost analysis, and a failure replay system for regression debugging.

🧩

Structured Binding

Schema-first library that forces LLMs into strict POJO structures and auto-repairs malformed JSON results.

🚦

Model Router

Smart gateway that load-balances requests across models (OpenAI/Claude/Local) based on cost and latency.
